Сервисы AI предназначены для транскрибации записей разговоров, анализа текстов звонков и чатов,
а также формирования кратких резюме. Стоимость использования определяется согласно [действующим тарифам](price.md).

### Предварительная подготовка
{.miko-man}
1. Для настройки параметров сервиса в панели разделов выберите [!badge Контакт-центр] :icon-chevron-right: [!badge Сервисы AI] :icon-chevron-right: [!badge Настройки сервисов AI].
2. Проверьте **текущий баланс**. При необходимости пополните баланс нажав кнопку [!badge Тарифы и пополнение].

## Анализ звонков

>>> Включите транскрибацию звонков
Для анализа звонков необходимо включить транскрибацию записей разговоров. Баланс расходуется отдельно на транскрибацию
и отдельно на анализ звонка.

1. В окне **Настройки сервисов AI** выберите [!badge Анализ звонков].
2. Включите опцию [!badge Переводить записи разговоров в текст].
3. Укажите минимальную и максимальную **длительность** записей разговоров, которые нужно обрабатывать.
Записи, не попадающие в указанный диапазон, обрабатываться не будут.

>>> Включите анализ звонков
1. Включите опцию [!badge Использовать нейросеть для анализа звонков].
2. Выберите [!badge Промпт].

В системе доступен предопределённый промпт для формирования краткого резюме.
При необходимости можно [создать собственный](promts.md).
>>>

## Анализ чатов

1. В окне **Настройки сервисов AI** выберите [!badge Анализ чатов].
2. Включите опцию [!badge Использовать нейросеть для анализа чатов].
3. Укажите минимальное **количество сообщений** в чате, начиная с которого будет выполняться анализ.
4. Выберите [!badge Промпт].

Чаты, в которых количество сообщений меньше заданного порога, анализироваться не будут.

## Регламентные задания

Анализ звонков и сообщений чатов выполняется по расписанию регламентного задания.

{.miko-man}
1. В панели разделов выберите [!badge Контакт-центр] :icon-chevron-right: [!badge Настройки] :icon-chevron-right: [!badge Регламентные задания].
2. Включите задание [!badge Обработка очереди документов].

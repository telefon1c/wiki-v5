В этом разделе задаются общие параметры работы телефонной подсистемы: правила обработки номеров, учет нерабочего времени
и поведение системы при вызовах.

#### Параметры телефонии

{.miko-man}
В панели разделов выберите [!badge Контакт-центр] :icon-chevron-right: [!badge Настройки] :icon-chevron-right: [!badge Настройки контакт-центра].

В группе [!badge Настройки телефонии] доступны следующие параметры.

Форматирование номеров телефонов
: Определяет, к какому виду приводятся телефонные номера перед сохранением в базе независимо от формата,
в котором они передаются провайдером. Можно выбрать национальный или международный формат номера.

Нерабочее время
: Используется при автоматической обработке обращений и исходящих звонков. Для исходящего вызова определяется местное
время абонента. Если на стороне абонента наступило нерабочее время, сотруднику выводится предупреждение с предложением
отказаться от звонка. Параметры нерабочего времени также могут использоваться для автоматических ответов в мессенджерах.

Действие при звонке
: Определяет алгоритм обработки вызовов.
- **Стандартные действия** — встроенные варианты обработки, которые пользователь может выбрать в персональных настройках.  
- **Свой алгоритм** — произвольный программный код, задаваемый администратором для обработки всех звонков.
  В этом случае пользователь не может изменить алгоритм обработки вызовов в персональных настройках.
  Недоступно в веб-клиенте.

Перехват вызова
: Определяет параметры автоматического перевода входящего вызова на сотрудника, ответственного за клиента.
Подробнее см. [Перехват вызова](call-interception.md).

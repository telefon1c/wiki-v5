Эта инструкция описывает процесс переноса истории звонков и сообщений из редакции 4 в новую подсистему контакт-центра.

### Предварительные требования
1. Выполните [первоначальную настройку](get-started.md).
2. Если требуется перенос сообщений, [подключите аккаунты мессенджеров](/administration/channels.md).

## Перенос данных

>>> Откройте сервис переноса данных
{.miko-man}
В панели разделов выберите [!badge Контакт-центр] :icon-chevron-right: [!badge Сервис] :icon-chevron-right: [!badge Перенос данных с редакции 4].

>>> Выберите какие данные будут перенесены

#### Переносить звонки
Дата начала
: Начало периода, за который будут перенесены звонки. Если не указана, будут выбраны звонки за весь доступный период,
начиная с самого раннего.

Дата окончания
: Конец периода, за который будут перенесены звонки. Если не указана, будут выбраны звонки за весь доступный период,
включая самые поздние.

Учетная запись АТС
: Учетная запись АТС в новой подсистеме, которая будет указана в реквизитах перенесенных звонков. 

#### Переносить сообщения
Дата начала
: Начало периода, за который будут перенесены сообщения. Если не указана, будут выбраны сообщения за весь доступный
период, начиная с самых ранних.

Дата окончания
: Конец периода, за который будут перенесены сообщения. Если не указана, будут выбраны сообщения за весь доступный
период, включая самые поздние.

Переносить групповые чаты
: Включите параметр, чтобы переносить не только личные диалоги, но и групповые чаты.

Переносить файлы и изображения
: Включите параметр, чтобы перенести вложения из сообщений. Учтите, что при использовании бесплатной версии
1С:Диалог срок хранения файлов и изображений ограничен. Если вложения уже удалены из сервиса, перенести их в новую
систему невозможно.

Соответствие учетных записей
: Укажите соответствие между учетными записями мессенджеров в редакции 4 и учетными записями в новой подсистеме.
Это необходимо для корректной привязки перенесенных сообщений к каналам связи.

Переносить чаты Telegram центра
: Включите параметр, чтобы перенести чаты Telegram центра в новую систему.

>>> Выполните перенос данных
